Résumé des commandes bash/shell

Commande Description
ls Afficher le contenu du dossier courant
Exemple 
x
C:\Repositories\Cours>ls
cpp/  git/  java/  lab/

 
cd <dir> Se déplacer dans le dossier <dir>
Exemple 
xxxxxxxxxx
C:\Repositories\Cours>cd git

      
C:\Repositories\Cours\git>cd ..

      
C:\Repositories\Cours>

 
mkdir <dir> Créer un dossier appelé <dir>
Exemple 
xxxxxxxxxx
C:\Repositories\Cours>mkdir tmp

      
C:\Repositories\Cours>ls
cpp/  git/  java/  lab/  tmp/

 
touch <file> Créer un fichier appelé <file>
Exemple 
xxxxxxxxxx
C:\Repositories\Cours>touch tmp.txt

      
C:\Repositories\Cours>ls
cpp/  git/  java/  lab/  tmp.txt

 
cat <file> Afficher le contenu d'un fichier appelé <file> 
xxxxxxxxxx
C:\Repositories\Cours>cat tmp.txt
Hello world
This content is here just for the example of the command cat
thank you

 
vim <file> Editer le contenu d'un fichier appelé <file>
Exemple 
xxxxxxxxxx
vim tmp.txt
