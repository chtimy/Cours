Boucles

Introduction

Une petite problématique, rien de telle pour commencer à expliquer un nouveau concept. Je souhaite créer un algorithme qui va me permettre de calculer le factoriel d'un nombre. Par exemple le factoriel de 5 qui s'écrit 5! se calcule en multipliant 5 par tous les nombres en dessous de lui, soit 5 x 4 x 3 x 2 x 1. Ce qui donne un résultat de 120.

Bien sûr on veut pouvoir le faire avec n'importe quel nombre. Il n'existe pas de solution naïve pour faire cette opération. On pourrait utiliser une fonction récursive qui s'appelle elle-même avec le résultat intermédiaire, mais c'est un autre concept que l'on a pas vu et il n'est pas forcément idéal dans notre cas, donc laissons de côté. On va plutôt s'intéresser à ce que l'on appelle une boucle. C'est un concept qui existe dans beaucoup de langages de programmations, car le système de boucle est quasiment indispensable.

Un petit tour avec la boucle for

La boucle for est un moyen d'exécuter du code plusieurs fois. On peut décider de combien de fois on veut exécuter le code en fonction d'une condition. Commençons par donner un exemple avant d'en dire plus. 

x
int main()
{
    for(auto i = 0; i < 5; i++)
    {
        std::cout << i << std:: endl; // Affiche 0 1 2 3 4
    }
    return 0;
}

La boucle for se décompose de la manière suivante :

  • En premier on écrit le mot clé for qui indique que l'on va exécuter une boucle à cet endroit là.
  • Ensuite viennent les () qui sont obligatoires. A l'intérieur des (), on trouve 3 champs séparés par des ;.
  • Le premier champs indique l'initialisation, dans le code ci-dessus, çà correspond à auto i = 0. On peut déclarer une variable dont la durée de vie sera celle de la boucle. On y créé généralement une variable qui va servir d'indice, son initialisation correspondra à l'indice de départ.
  • Sautons directement au troisième champs qui correspond à une ou plusieurs instructions qui seront exécutées à chaque fin de boucle. Dans le code ci-dessus, on incrémente l'indice i de 1 à chaque fois. Si tu n'es pas familier avec i++, c'est la même chose que d'écrire i = i + 1 ou encore i += 1. On a donc un indice qu'on peut initialiser dans le premier champs et qu'on peut modifier dans le 3ème à chaque tour de boucle.
  • Il faut maintenant savoir quand est-ce qu'on arrête la boucle et c'est le rôle du second champs. Dans ce champs, on peut écrire une ou plusieurs conditions qui indiquent si l'on doit rester dans la boucle. Si la ou les conditions ne sont pas respectées alors on sort de la boucle. Dans le code ci-dessus, on a écrit i < 5, ce qui signifie que lorsque i dépassera 4, la condition n'est plus respectée et on continue l'exécution du code qui se situe après la boucle for.
  • L'entête de la boucle for est terminé, il ne reste plus qu'à écrire le contenu de la boucle qui va se situer entre {} immédiatement après l'entête. Le code est exécuté autant de fois que la condition de la boucle for est vrai.

En d'autres mots, on pourrait lire la boucle for de la manière suivante : Pour i allant de 0 à 4 faire .... C'est plus lyrique et plus facile de se représenter la boucle for de cette manière.

Il faut savoir que les 3 champs sont optionnels. Essayons de jouer avec çà dans le code suivant: 

x
#include <iostream>
int main()
{
    // On initialise i à l'extérieur de la boucle for
    int i = 0; 
    for(; i < 5; i++)
    {
        std::cout << i << std:: endl; // Affiche 0 1 2 3 4
    }
    // On peut réutiliser i après cette boucle
    std::cout << i << std::endl;
    
    // Il n'y a pas de condition dans la boucle for, le code considère ici que la condition est toujours vrai. C'est donc une boucle infinie
    for(int i = 0; ; i++)
    {
        std::cout << i << std:: endl; // Affiche 0 1 2 3 4
    }
    // Dans ce cas, i n'est jamais incrémenté et c'est encore une boucle infinie
    for(int i = 0; i < 5; )
    {
        std::cout << i << std:: endl; // Affiche 0 infiniment
    }
    
    // Ici, aucune initialisation ni instruction à exécuter à chaque tour de boucle. Comme la condition est toujours vrai, la boucle est encore une boucle infinie
    for(; ; )
    {
    }
    
    // Il est tout à fait possible d'initialiser plusieurs variables, de composer plusieurs conditions et de réaliser plusieurs instructions dans une même boucle for
    for(int i = 0, j = 0; i < 5 && j < 3; i++, j++)
    {
        std::cout << i << std:: endl; // Affiche 0 1 2
        std::cout << j << std:: endl; // Affiche 0 1 2
    }
    
    return 0;
}

C'est à peu près tout ce qu'il y a à dire sur la boucle for. Il te reste maintenant plus qu'à écrire le code pour réaliser le factoriel d'un nombre. Une fois que tu auras essayé, tu peux vérifier en comparant ta solution avec celle que je propose ci-dessous (déplie le passage pour voir la solution).

Solution 
x
int factoriel(int value)
{
    auto total = value;
    for(auto i = value - 1; i > 1 ; i--)
    {
        total *= i;
    }
    return total;
}
